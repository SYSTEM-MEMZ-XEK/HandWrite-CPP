# HandWrite Generator (C++)

> 基于 Qt6 的手写字生成器，通过自然扰动渲染文本以模拟手写效果

这是 [HandWrite](../) 项目的 C++ 移植版本，提供更快的渲染性能和更小的内存占用。

## 功能特性

- **纸张设置**：自定义纸张宽度和高度
- **字体选择**：支持自定义 TTF 字体文件
- **字体大小**：自由调整字间距大小
- **行距字距**：行距和字距调整
- **留白设置**：上下左右边距设置
- **颜色选择**：字体颜色和背景颜色（支持透明）
- **扰动模拟**：行间距扰动、字体大小扰动、字间距扰动、横向/纵向笔画扰动、旋转扰动
- **分页预览**：长文本自动分页，支持预览和页码导航
- **缩放预览**：支持预览图像缩放
- **配置管理**：保存/加载配置文件
- **字符覆盖**：对选定字符范围应用自定义属性
- **并行渲染**：多线程并行生成，提升效率
- **高清导出**：支持多种分辨率倍率导出

## 构建要求

- CMake >= 3.16
- C++17 编译器
- Qt6 (Core, Gui, Widgets, Concurrent)

## 构建步骤

```bash
cd HandWrite-CPP
mkdir build && cd build
cmake ..
cmake --build . --config Release
```

构建完成后，可执行文件位于 `build` 目录下，`ttf_library` 字体文件夹会自动复制到构建目录。

## 运行

```bash
./HandWrite.exe
```

或直接双击运行 `HandWrite.exe`。

## 使用说明

### 基本参数

| 参数 | 说明 |
|------|------|
| 宽度/高度 | 纸张尺寸（像素） |
| 字体 | 选择 TTF 字体文件 |
| 字体大小 | 字体大小（像素） |
| 行距 | 行与行之间的间距 |
| 字距 | 字符之间的间距 |
| 上/下/左/右边距 | 纸张边缘留白 |

### 扰动参数

通过调整扰动标准差模拟自然手写效果：

| 参数 | 说明 |
|------|------|
| 行间距扰动 | 行距随机变化 |
| 字体大小扰动 | 字体大小随机变化 |
| 字间距扰动 | 字符间距随机变化 |
| 横向扰动 | 字符水平位置偏移 |
| 纵向扰动 | 字符垂直位置偏移 |
| 旋转扰动 | 字符旋转角度 |

扰动值越大，效果越潦草；设为 0 可得到整齐的输出。

### 颜色设置

- **字体颜色**：支持 RGBA 设置
- **背景颜色**：支持透明背景（alpha = 0）或纯色背景

### 导出

点击 Export 按钮导出图片，默认输出到 `outputs` 目录。支持选择导出倍率，倍率越高图像越清晰。

### 配置文件

支持保存和加载 TOML 格式的配置文件，方便复用参数设置。

## 项目结构

```
HandWrite-CPP/
├── CMakeLists.txt      # CMake 构建配置
├── src/
│   ├── main.cpp        # 程序入口
│   ├── config.hpp/cpp  # TOML 配置解析器
│   ├── tools.hpp/cpp   # 工具函数
│   ├── core.hpp/cpp    # 核心生成引擎
│   └── mainwindow.hpp/cpp  # Qt 主窗口
├── ui/
│   └── mainwindow.ui   # Qt Designer 界面文件
└── resources/
    ├── app.ico         # 应用图标
    └── app.rc.in       # Windows 资源模板
```

## 核心类

- `HandwriteGenerator`：手写生成引擎，负责文本布局和渲染
- `TemplateParams`：模板参数结构体
- `MainWindow`：Qt 主窗口类
- `Config`：TOML 配置解析器
